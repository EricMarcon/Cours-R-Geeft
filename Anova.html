Il faut utiliser le test de Kruskal-Wallis, qui est un modle linaire sur les rangs.

Le problme des tests multiples

Le seuil de risque de 5% signifie que 5% des tests seront des faux positifs.

Avec 7 groupes, on fait \(6 \times 7 / 2 = 21\) tests dgalit, donc on attend un faux positif.

Solution : rduire le seuil de risque \(\alpha_m\) (pour multiple) : \[\alpha_m = 1 - (1 - \alpha)^n \approx \alpha / n\] Cest la correction de Bonferroni.
